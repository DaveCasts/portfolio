Temperature Conversion

This Ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 


  			puts "Enter a temperature in celsius: "

			temp = gets.chomp

			fahr = (temp.to_i * 9 / 5) + 32

			puts "Your temperature converted to fahrenheit is #{fahr}."
		


  			puts "Enter a number: "

			num = gets.chomp.to_i

			ones_place = num % 10
			tens_place = num % 100

			if tens_place >= 11 && tens_place <= 19
			    puts "That's the #{num}th item."
			elsif ones_place == 0 
			    puts "That's the #{num}th item."
			elsif ones_place == 1
			    puts "That's the #{num}st item."

			elsif ones_place == 3
			    puts "That's the #{num}rd item."
			else
			    puts "That's the #(num}th item."
			end

Foobar

This Ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 


			puts "How many items do you want to see? "

			num = gets.chomp.to_i

			[*1..num].each do |x|

			    div_by_5 = x % 5 == 0
			    div_by_3 = x % 3 == 0

			    if div_by_3 && div_by_5
			        puts "FooBar"
			    elsif div_by_3
			        puts "Foo"
			    elsif div_by_5
			        puts "Bar"
			    else 
			        puts "#{x}"

			    end
			end
